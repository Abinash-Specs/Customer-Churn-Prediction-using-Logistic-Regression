\# 📊 Customer Churn Prediction (Advanced) using Machine Learning



\## 📌 Overview



This project focuses on predicting customer churn using advanced machine learning techniques. Customer churn refers to customers discontinuing a service, which is a critical business problem affecting revenue and growth.



Using real-world telecom customer data, this project builds and evaluates predictive models to identify customers who are likely to churn.



\---



\## 🎯 Objectives



\* Perform data preprocessing and cleaning

\* Conduct in-depth exploratory data analysis (EDA)

\* Handle feature encoding and transformations

\* Train and compare multiple classification models

\* Evaluate model performance using various metrics

\* Identify key factors influencing customer churn



\---



\## 📂 Dataset Description



The dataset contains telecom customer information including:



\* Account details

\* Service usage patterns

\* Customer demographics

\* Churn status (target variable)



Two datasets are used:



\* Training dataset

\* Testing dataset
